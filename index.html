GRID: Flexbox con Esteroides

Orígenes:

Diseño gráfico: las Grillas gráficas


El Grid o Grillas, o Cuadrillas, se utiliza ampliamente en areas como Arquitectura, Diseño Industrial, Diseño Gráfico, o cualquier area relacionada.
El Grid se utiliza principalmente para el encuadre exacto de algun trazado gráfico, ya sea para planos de un edificio, el diseño de un producto, el diseño de un logo, incluso para ilustraciones o revistas.

El desarrollo Web maduró al punto de necesitar nuevas técnicas de diseño que permita que el maquetado de una página sea visualmente accesible desde cualquier dispositivo.


Flexbox fué toda una revolución a la hora de maquetar páginas web, la técnica consiste en utilizar un modelo de caja unidimensional.


Flexbox


En cambio, el Grid utiliza un modelo de caja bidimensional, podemos utilizar Columnas y Filas.
